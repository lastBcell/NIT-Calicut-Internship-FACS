# FACS Tools & Libraries

| Tool | Purpose | Main Features |
|---|---|---|
| OpenFace | Facial behavior analysis toolkit | AU detection, gaze tracking, landmark detection, head pose estimation |
| NetFACS | FACS network analysis | Studies relationships between Action Units using graph/network analysis |
| PyAFAR | Python AU recognition library | Automated AU detection for adults and infants |
| OpenFace-FACS-Unity-Facial-Animator | Real-time avatar animation | Streams AU values from OpenFace into Unity |
| openFACS | 3D facial animation system | Generates realistic facial animation using FACS |
| pyfaceau | Pure Python OpenFace implementation | AU extraction without C++ compilation |
| LibreFace | Deep learning facial analysis toolkit | AU detection, AU intensity estimation, FER |
| OpenFaceR | R analysis toolkit for OpenFace outputs | Statistical analysis of AU CSV outputs |
| exploface | OpenFace output analysis | Visualization and analysis of OpenFace CSV outputs |
