# Additional FACS / AU Tools

| Tool | Purpose | Description |
|---|---|---|
| Py-Feat | Facial expression analysis | Python toolkit for AU, emotion, landmarks, and gaze |
| FeatLab | AU analysis platform | GUI-based facial analysis workflow |
| DeepFace | Deep facial analysis | Emotion, age, gender, verification |
| InsightFace | Face recognition framework | Powerful face embeddings and alignment |
| MediaPipe Face Mesh | Landmark tracking | Real-time 468 facial landmarks |
| FaceAlignment | Landmark estimation | 2D/3D facial landmark detection |
| EMOCA | Emotional 3D face reconstruction | Generates expressive 3D faces |
| DECA | Detailed face reconstruction | Captures wrinkles and detailed expressions |
| OpenSeeFace | Real-time face tracking | VTuber and avatar facial tracking |
| FaceFormer | Audio-driven face animation | Speech-to-face animation transformer |
